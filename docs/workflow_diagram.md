## Key Takeaway

**Semi-automatic mode is NOT fully automatic!**

You must remember to:
1. Open Kiro after each commit
2. Ask Kiro to fix the drift

If you forget this step, the drift will remain unfixed.

**Recommendation:** Use task mode if you want more control, or blocking mode if you want to ensure zero drift.
